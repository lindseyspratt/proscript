ProscriptLS Download

The ProscriptLS project is https://github.com/lindseyspratt/proscriptls .
The ProscriptLS library is
https://github.com/lindseyspratt/proscriptls/blob/master/dist/proscriptls.js .
This library is available in two pieces, proscriptls_engine.js and proscriptls_state.js .
There is a specialized form of the main library for compiling source to create custom state, proscriptls_for_compile.js , and a node script to run the compilation, node_compile.js .
