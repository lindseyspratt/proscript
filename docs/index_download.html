Downloading ProscriptLS

The main ProscriptLS library is
https://github.com/lindseyspratt/proscript/blob/master/dist/proscriptls.js
The main library has two components, the engine and the initial program state. You may compile your own initial program state using the ProscriptLS wam_compiler. Loading pre-compiled ProscriptLS programs can be a large performance improvement over having a web page compile your ProscriptLS source. The engine is
https://github.com/lindseyspratt/proscript/blob/master/dist/proscriptls_engine.js
The (default) ProscriptLS initial program state is
https://github.com/lindseyspratt/proscript/blob/master/dist/proscriptls_state.js
