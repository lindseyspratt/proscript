About ProscriptLS

ProscriptLS is developed by Lindsey Spratt (Lindsey dot Spratt at LogicForSystems dot com). It is a fork of the original Proscript project by Matt Lilley.
There are three major enhancements and many minor changes in ProscriptLS:
  • builtins to work with the DOM (Document Object Model) of HTML,
  • an interactive Prolog tracing debugger that can be used in the client web page (using a JQuery Terminal), and
  • support for modules.

The original Proscript project implemented the core of the system, particularly the WAM, the WAM compiler, and the basic architecture of the system.
Version 1.2.2, October 2019. Powered by ProscriptLS Prolog - Logic at Your Service.
