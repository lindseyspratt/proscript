About ProscriptLS

ProscriptLS is developed by Lindsey Spratt (Lindsey dot Spratt at LogicForSystems dot com). It is a fork of the original Proscript project by Matt Lilley.
There are two major enhancements and many minor changes in ProscriptLS: builtins to work with the DOM (Document Object Model) of HTML and a proper interactive Prolog tracing debugger that can be used in the client web page (using a JQuery Terminal).
The original Proscript project implemented the bulk of the system, particularly the WAM, the WAM compiler, and the basic architecture of the system.
Version 1, June 2019. Powered by ProscriptLS Prolog - Logic at Your Service.
